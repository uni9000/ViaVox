# ViaVox: 한국어 음성 및 화자 인식 기반 음성 노트

## 프로젝트 개요

ViaVox는 딥러닝을 활용하여 **한국어 음성 인식** 및 **화자 인식** 기능을 제공하는 음성 노트 애플리케이션입니다. 사용자는 음성을 녹음하고 이를 텍스트로 변환(STT)할 수 있으며, 개별 화자를 구별하고 대화 내용을 요약하는 기능을 제공합니다.

## 주요 기능

- **음성 인식(STT)**: 사전 학습된 모델을 사용하여 음성을 텍스트로 변환.
- **화자 인식**: 개별 화자를 구분하여 발언자를 인식.
- **대화 요약**: OpenAI GPT API를 사용하여 대화 또는 회의 내용을 요약.
- **자동 회의록 작성**: 여러 화자의 발언을 인식하여 자동으로 회의록을 작성.

- **models/**: 음성 및 화자 인식 모델 파일.

- **package.json**: 프로젝트에 필요한 NPM 패키지 목록.

## 설치 방법

### 요구 사항

- Python 3.8 이상
- Node.js 14 이상
- Pytorch 또는 Tensorflow
- OpenAI GPT API Key

### 설치 절차

1. **리포지토리 클론**
```bash
   git clone https://github.com/yourusername/viavox.git
   cd viavox
```
2. **Python 패키지 설치**
```bash
  pip install -r requirements.txt
```
3. **Node.js 패키지 설치**
```bash
  npm install
```
4. **GPT API 키 설정 .env 파일을 생성하고, OpenAI GPT API 키를 추가하세요.**
```bash
  OPENAI_API_KEY=your-api-key
```

## 사용법

1. **음성 녹음 시작**
   - 앱의 메인 화면에서 "음성 기록 시작하기" 버튼을 눌러 녹음을 시작합니다.
   - 녹음을 마치려면 "녹음 중지" 버튼을 누릅니다.
   - 녹음이 완료되면 자동으로 텍스트로 변환되며, 해당 텍스트는 화면에 표시됩니다.

2. **화자 인식**
   - 녹음된 음성에서 다수의 화자가 있는 경우, 앱은 각 화자를 구별하여 화자별로 라벨을 붙입니다.
   - 화자가 바뀔 때마다 화면에 화자의 이름이 표시됩니다(사전에 화자 정보가 등록되어 있어야 함).

3. **대화 요약**
   - 녹음이 완료된 후, "대화 요약하기" 버튼을 눌러 대화 내용을 요약합니다.
   - 요약된 내용은 화면에 표시되며, 필요에 따라 수정할 수 있습니다.

4. **설정 변경**
   - 사용자는 설정 메뉴를 통해 음성 인식, 화자 인식 설정 및 기타 옵션을 조정할 수 있습니다.
   - 설정에서 API 키를 업데이트하거나, 음성 인식의 민감도를 조절할 수 있습니다.
